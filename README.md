Zones of Thought Strategy Game

## Pre-requisite

- JDK 11
- Clojure 1.10.1
- `clojure` command-line tool 1.10.1.727 or later

## Running

From CLI run the below command:

Windows:

```
cp deps-win.edn deps.edn
clj -M -m zot.main
```

MacOs:

```sh
cp deps-mac.edn deps.edn
clj -J-XstartOnFirstThread -M -m zot.main
```
